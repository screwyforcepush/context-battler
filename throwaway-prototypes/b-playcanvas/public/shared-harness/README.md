# Shared Replay Harness

This directory is the static shared input for the throwaway render
prototypes. It intentionally contains JSON and fixed sourced assets only:
no live Convex wiring, no runtime abstraction, and no production art
pipeline.

## Replay Snapshot

`replay-snapshot.json` is generated by:

```bash
node throwaway-prototypes/shared-harness/tools/generate-replay-snapshot.mjs
```

The generator reads `maps/reference.json` and writes a deterministic
fixture whose `timeline.frames[].snapshot` objects follow
`apps/replay/src/lib/reconstruct.ts#EntitySnapshot`.

Fixture summary:

- Slice: turns 7 through 12, 16 seconds.
- Map: `reference`, including size, walls, cover clusters, static crates,
  airdrops, and the evac zone.
- Event: `Crate_50_50` is telegraphed on turns 7-10 and lands on turn 10.
- Victim: `char_sprinter` / Sprinter moves from `{ "x": 49, "y": 50 }`
  onto `{ "x": 50, "y": 50 }` on turn 10.
- Result: turn 10 has `environmentalDeaths: ["char_sprinter"]`,
  `deaths: []`, and `corpses: []`. Turn 11 shows the airdrop as landed.

The fixture keeps HP and equipment as `null`, matching the reconstruct
contract: those fields are not derivable from the turn ledger in replay
v0.

## Art Kit

`art-kit/` is a minimal fixed subset of Quaternius' Ultimate Space Kit:

- `Astronaut.glb` for agents.
- `Pickup Crate.glb` for the airdrop crate.
- `Base Large.glb` and `Building L.glb` as lightweight environment props.
- `manifest.json` records source URLs, license, sizes, and SHA-256 hashes.

Source and license:

- OpenGameArt page: https://opengameart.org/content/ultimate-space-kit-by-quaternius
- OpenGameArt GLB zip: https://opengameart.org/sites/default/files/ultimate_space_kit-glb.zip
- Quaternius pack page: https://quaternius.com/packs/ultimatespacekit.html
- License: CC0 1.0 Universal / Public Domain, https://creativecommons.org/publicdomain/zero/1.0/

The local extraction used `/tmp/context-battler-assets/ultimate_space_kit-glb.zip`.
If the archive is missing, recreate it with:

```bash
mkdir -p /tmp/context-battler-assets
curl -L "https://opengameart.org/sites/default/files/ultimate_space_kit-glb.zip" \
  -o /tmp/context-battler-assets/ultimate_space_kit-glb.zip
```

This is a throwaway sourced kit for substrate R&D. It is not production
art, and the selected files should stay fixed so later prototypes render
the same scene bytes.
