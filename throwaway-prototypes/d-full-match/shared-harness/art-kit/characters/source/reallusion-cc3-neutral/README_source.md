# Reallusion CC3 Neutral Template Source

This directory contains the official Reallusion CC3 neutral FBX body template used for the `experiment` head-transplant spike.

Source page:

```text
https://wiki.reallusion.com/Content_Dev:CC_Asset_Provision
```

Direct source package used:

```text
https://wiki.reallusion.com/images/9/91/Base_Neutral_FBX.rar
```

The account-gated newer Reallusion "Free 3D Character Base" package was not accessible from this headless environment. This older official template is used to prove the Mesh2Motion body + Reallusion head topology transplant path. It has real CC head topology and separate eyes/teeth/tongue, but this package did not import with facial shape keys in Blender.
